# 🔐 Password Strength Checker

A Python-based tool to check the strength of passwords based on security rules.

---

## 🚀 Features
- Checks password strength
- Validates length and complexity
- Detects weak passwords

---

## 🛠️ Technologies Used
- Python

---

## ▶️ Usage

Run the program:

python main.py

---

## 📸 Output Example

Password: hello123  
Strength: Weak ❌  

Password: H@ck3r!2024  
Strength: Strong ✅  

---

## 📌 Project Description (For Resume)

Developed a Python-based password strength checker that evaluates password security using length, character variety, and pattern validation.
